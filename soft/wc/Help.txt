Wild Commander v1.00

Управление:

В редакторе строки:
CTRL+SHIFT - смена раскладки ENG/RUS
INSERT - переключение режима INS/OVR
CAPS LOCK - включение/выключение режима CAPS

В панелях:
ALT+F1/F2 - выбор драйвера в соотв. панели
TAB - смена панели
SPACE - маркировать файлы/каталоги (дублируется клавишей INSERT)
F2 - открыть меню пользователя (если был загружен плагин). задается в wc.mnu (в папке с плагинами)
F4 - редактировать файл под курсором в текстовом редакторе (если был загружен плагин)
F5 - копировать отмеченные файлы/каталоги в противоположную панель (выводит диалоговое окно)
F6 - переименовать файл/каталог под курсором (время создания не меняет)
F7 - создать каталог в активной панели
F8 - удалить отмеченные файлы/каталоги (дублируется клавишей DEL)
F9 - настройка INI файла (автоматически создает папку WC на boot-device, если таковой нету)
F10 - вызов меню запуска/удаления плагинов. по ENTER таковые запускаются (если поддерживают запуск из меню), по DEL удаляются из памяти.

ctrl+F3 - включение/выключение сортировки по имени
ctrl+F4 - включение/выключение сортировки по расширению
ctrl+F6 - включение/выключение сортировки по размеру
ctrl+R - перечитать содержимое каталога из панели
ctrl+A - выделить все элементы в каталоге

* - инвертировать маркировку в каталоге

ENTER - войти в каталог/загрузить WMF плагин/запустить SPG/запустить $C/просмотреть файл под курсором
SHIFT+ENTER - вывод меню выбора вьювера (формируется из встроенных вьюверов и загруженных плагинов, поддерживающих данное меню)
F3 - открыть файл на просмотр (как текст/HEX/картинку (TGA/SCR/Gigascreen/MGS). загружает файлы до 1мб. Если расширение не опознано, то будет предложено выбрать действие


P.S. во время операций удаления/копирования файлов/каталогов можно досрочно прервать таковые нажатием на ESC (меню появится на смене файла)!
